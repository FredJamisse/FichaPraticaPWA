1 - O que são operações de “Commit”, “Push” e “Pop” efetuadas num repositório?
Resposta: “Commit” é uma operação que serve para guardar as alterações feitas no código, essas alterações sempre vão com uma descrição.
“Push” é uma operação que serve para enviar o código para o repositório online
“Pop” é uma operação que serve para guardar as alterações localmente antes de commit e push. 
2 – Quais sáo as vantagens de usar Branchs num repositório git?
Resposta: Usar Branches num repositório git é vantajoso porque permite fazer alterações sem comprometer o código principal.
